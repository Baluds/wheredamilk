FAQ

What does WhereDaMilk do?

WhereDaMilk is a voice-controlled AI vision assistant. It uses your webcam to detect objects, read text on labels, and give you spoken directions to find items ‚Äî completely hands-free.

Do I need a GPU?

No. WhereDaMilk runs on CPU. A GPU will make it faster, but it works fine without one.

What hardware do I need?

A webcam and a microphone. Built-in laptop webcam and mic work perfectly.

How does voice control work?

WhereDaMilk listens through your microphone using Google Speech Recognition. Just say commands like "find milk" or "read" and it responds in real-time.

What text-to-speech engine is used?

It uses ElevenLabs for high-quality TTS with an API key. If unavailable, it falls back to Microsoft Edge TTS which is free and requires no setup.

Can it detect any object?

It uses YOLOv8 which can detect 80+ common object classes (bottles, cups, phones, people, etc). For text-based items like specific brands, it uses OCR to read the label text.
